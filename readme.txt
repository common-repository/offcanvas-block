Sidebar drawer that can be toggled via a button. 

== Description ==

OffCanvas is a sidebar drawer that can be toggled via a button to appear from the left, right, or bottom edge of the viewport.


**[Demos](https://bblockswp.com/demo/offcanvas-block/)**


= Features =
- **Position**: Set OffCanvas position(left, right, top and bottom).
- **Button Style**: Apply button align, typography and colors(text and background).
- **Popup Style**: Add background property such as color, gradient or image.
- **Popup Header Style**: Apply header title typography and color. Also, color in the close icon.
- **InnerBlocks**: InnerBlocks in the popup area for custom content as per your requirements.
- **Library Use**: Bootstrap, React Bootstrap.


= How to use =
- First, install the OffCanvas Block plugin
- Add the OffCanvas Block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **OffCanvas Block**
3. Click on the **OffCanvas Block** to add the block

= Download & Upload: =
1. Download the **OffCanvas Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`offcanvas-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **OffCanvas Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
